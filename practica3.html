Equipo #15

¿Qué es CSS?

CSS (en inglés Cascading Style Sheets) es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML. CSS separa el contenido de la representación visual del sitio. CSS fue desarrollado por W3C (World Wide Web Consortium) en 1996 por una razón muy sencilla. HTML no fue diseñado para tener etiquetas que ayuden a formatear la página. Está hecho solo para escribir el marcado para el sitio. La relación entre HTML y CSS es muy fuerte. Dado que HTML es un lenguaje de marcado (es decir, constituye la base de un sitio) y CSS enfatiza el estilo (toda la parte estética de un sitio web), van de la mano.

¿Para qué sirve CSS?

CSS es un lenguaje utilizado en la presentación de documentos HTML. Un documento HTML viene siendo coloquialmente “una página web”. Entonces podemos decir que el lenguaje CSS sirve para organizar la presentación y aspecto de una página web. Este lenguaje es principalmente utilizado por parte de los navegadores web de internet y por los programadores web informáticos para elegir multitud de opciones de presentación como colores, tipos y tamaños de letra, etc.

¿Cuál es la diferencia que sería más difícil de realizar en un sitio web si se optara por no usar un CSS?

CSS permite dar una estructura a la página web para que sea fácil de navegar. Mientras que los colores no afectan a la funcionalidad, una página sin una buena estructura de diseño y que posicione los elementos de manera correcta sería casi imposible de utilizar. Si se opta por no usar CSS, la lógica para manejar los tamaños y posiciones de los elementos debería ser codificada en JavaScript y aumentaría la complejidad del código.

Algunas opciones básicas del lenguaje CSS por ejemplo pueden ser el poder cambiar el color de algunas típicas etiquetas HTML como <H1>; pero también hay funciones algo más complejas, como introducir espaciado entre elementos <DIV> CSS es muy intuitivo y sencillo una vez se llega a aprender, ya que te puedes apoyar en el uso de los identificadores de etiqueta HTML, luego indicamos con qué aspecto queremos que se muestren todas las etiquetas, y listo. Para crear el archivo basta abrir un editor de texto como el bloc de notas escribir el texto y guardar el archivo con el nombre adecuado.

Reset CSS

Todos los elementos de HTML tienen estilos por defecto dependiendo del navegador, pero gracias a CSS se le puede cambiar cualquier tipo de cosa a la pagina, el tamaño de la letra, la fuente, los margenes, etc., y hay una tecnica para poder eliminar todos los estilos por defecto a los elementos HTML, lo que setiene que hacer es agregarle en la cabeza del HTML un CSS el cual tenga de nombre ¨reset.css¨ creando asi una hoja la cual prefija los margenes, las distancias a 0.

Historia de las Técnicas de presentación en pantalla para la Web

Se empezó a dar diseño a las webs a mediado de los 90 el único método que se podía usar para crear estructuras visuales complejas era usar el elemento tabla (<table>), a partir de esto se empezó a usar técnicas como anidamiento de tablas en la cual juntaban varias tablas así creando varios espacios en los cuales ponían información, fotos, etc, ahí dentro, dando así un estilo visual único en ese tiempo.

Distintas opciones de presentación de páginas

Al momento de crear la estructura visual de la pagina se tienen que saber varias cosas, por ejemplo, la anchura de esta, hay dos categorías principales de estructura en base a la anchura las cuales son: las de anchura fija y las de ancho variable.

En las estructuras de ancho fijo como su nombre lo dice se mantiene dentro del ancho que se le especifico, se le puede poner el valor en pixels pero comúnmente la unidad que se usa es em, esta estructura es cómoda para el diseñador ya que le deja poner todas las cosas precisamente donde las quiere (cabecera, barras laterales, pies de paginas, etc), incluso da un esqueleto firme para otros elementos como la anchura de los párrafos, el posicionamiento de las imágenes, etc.

Del otro lado tenemos las estructuras flexibles las cuales son hechas para adaptarse a la anchura de la ventana del navegador en cualquier momento, estas estructuras son mas recomendables que las fijas ya que hoy en día hay muchas pantallas de diferentes tamaños y no tan solo es lapota y computadoras, ahora hay gran uso de celulares, tabletas, etc., por lo cual son muy adecuadas debido a que permiten re posicionar los textos e imágenes para mayor comidan del usuario, pero tienen su desventaja la cual es que son mas difíciles de diseñar.

El elemento <div>

Se le puede considerar al elemento <div> como un contenido genérico en el que se pueden poner elementos que se relacionen entre si de forma lógica. Los elementos <div> suelen tener un ID o una pertenecer a una clase para asi poder darle estilos CSS, esto facilita el identifica las diferentes secciones de la pagina y poder aplicar el estilo que queramos ahí.

La propiedad float CSS

Esta propiedad permite que se puede escribir alrededor de una imagen, ya que cambia la posición relativa con respecto al elemento flotante. En los elementos float se le pueden dar valores ¨left¨, ¨right¨y ¨none¨ para dirigir en donde se pondría el texto. Se pueden colocar varios elementos flotantes dentro del mismo contenedor haciendo que todos se alinean.

La propiedad clear

Esta propiedad hace que no haya elementos flotantes en los lados del objeto, se le puede especificar que no se ponga en la izquierda, en la derecha o en los dos lados.
