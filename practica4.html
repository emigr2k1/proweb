Lección 2: Conceptos básicos de HTML, XHTML y CSS

La estructura de la página web se basa en HTML Los documentos de HTML tienen la extension ,htm o .html, esta extensión avisa al navegador web ya sea de una computadora o de un dispositivo movil que el archivo es HTML y este se muestra en la ventana del navegador siguiendo las reglas del lenguaje HTML.

HTML hace uso de etiquetas para definir el contenido con el que trabaja, las etiquetas estan dentro del ¨menor que¨ (<) y el ¨mayor que¨ (>) por ejemplo <p></p>, estas no se ven en el navegador pero dentro de él le da la propiedad de parrafo a lo que este dentro de las etiquetas.

Detalles de la sintaxis de XHTML Las paginas en XHTML y HTML tienen los mismos atribuos y etiquetas pero tienen una diferencia importante el cual es que la sintaxis del código en XHTML es más estricta, por ejemplo: -Las eqtiquetas en XHTML deben escribirse en minusculas. -Todas las etiquetas que se abran deben de cerrarse obligatoriamente. -El anidamiento en las etiquetas tiene ciertas reglas, por ejemplo, el <em> debe abrirse despes del <h1> y cerrarse antes del </h1>

Doctype indica al navegador qué debe de esperar En toda pagina web debe tener al principio declarado el Doctype, ya que le aporta pistas al navegador sobre el tipo de informacion que va a haber en el página, debido a que HTML y XHTML son diferentes el navegador necesita saber cual es el lenguaje que va a interpretar, el navegador lee la página desde el principio, asi hasta el final, se escribe de esta manera: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

El W3C y la validación de la pagina W3C es la abreviatura de World Wide Web Consortium, el cual es una organizacion que busca orientar la evolución de la Web, esta ofrece reglas y directrices para especificaciones como HTML y XHTML. Para poder verifica la validez del codigo HTML o XHTML se puede usar el servicio online gratuito que ofrecen.

Estructura HTML En HTML la estructura que se presenta es anidada, los contenidos estan unos dentros de otro, por ejemplo: <html> <body> </body> </html> El elemento body esta dentro del elemeno html, este se abre dentro del html y se cierra antes, estos elementos son la base para toda pagina web. Lo que se encuentra dentro del body es el contenido el cual va a ser visible para los usuarios, hay otras partes las cuales son ocultas por ejemplo el script el cual permite darle interatividad a la página o también el código que permite darle estilo a la página, estos comunmente van dentro del <head>, el cual esta dentro del <html>.

Insertar imágenes en HTML Se hace uso de una etiqueta para poder poner una imagen en HTML la cual es <img>, esta en sí no hace nada todo depende del valor que le demos a los atributos, los cuales indican que imagen mostrar y como se debe hacer, ejemplo: <img src="images/familysitting.jpg" />

El papel de CSS CSS son las siglas de Cascading Style Sheets, estas tienen un lenguaje diferentes a HTML y nos permiten aplicarle estilos a los diferentes elementos que tengamos en la página web, haciendo que todos los títulos, listas, parrafos, etc. puedan verse igual en todas las páginas que tengamos.

Los estilos de clase y el elemento <span> Los selectores de etiqueta son muy usados frecuentemente pero solo se pueden aplicar a los elementos HTML, pero cuando queremos modificar el estilo de algo que no coincida con un etiqueta, por ejemplo cambiar solo una palabra de un párrafo, no son de mucha ayuda los selectores de etiqueta, lo que se utiliza son los selectores de clase, la cual es una regla CSS que puede aplicarse a cualquier elemento dentro de la página, estos tienen la opcion de poner nombres flexibles pero es recomendable usar nombres los cuales describan bien lo que hacen.

Tres maneras de utilizar estilos Hay tres maneras diferentes de incorporar los estilos CSS a la página, los cuales son: Hoja de estilos interna: Este es cuando especificas los estilos dentro del <head>.

Hoja de estilo externa: Es un documento que esta fuera del HTML el cual tiene una extensión .css, para que pueda aplicarse a la página se le debe de poner un enlace al archivo dentro del <head> del HTML.

Estilos inline o directos: Las reglas se declaran dentro de las etiquetas de HTML por lo cual son las menos usadas.

Cuándo utilizar hojas de estilo internas y externas Cuando tienes muchas páginas en tu sitio web es recomendable el usar un estilo de hoja externa para así tener todas las páginas con un mismo orden y cuando decidas cambiar algo solo vas a ese archivo, lo modificas y automaticamente modifica todas las páginas.

Creación de una hoja de estilos externa Las páginas HTML no tienen por que límitarse a usar solamente una hoja de etilos externa, se pueden tener varias, ya sea para cuando se va a imprimir una pagina o para poder ver la página de manera perfecta en los dispositivos móviles.

Uso de textos y tipos de letra en la web Cuando se diseña una págnina web se puede cambiar el estilo de texto que tiene por defecto el navegador, pero el usuario debe de tener descargada esa fuente, si no la tiene el navegador le pondra otra, una solucion para eso es el ¨font-family¨ en el cual pones una lista de las fuentes de letra que quieres que se visualicen y el navegador selecciona una el cual usario tenga descargada.

Tambien se le puede cambiar el tamaño al texto gracia a CSS con la propiedad ¨font-size¨ el cual se puede cambiar el valor de diferentes maneras: -Tamaño absoluto: Son palabras las cuales tienen tamaños ya predefinidos, po ejemplo, xx-small, medium,x-large, etc.

-Longitud: En este puedes indicar un sistema de medida ya sea cm, mm, in, pt o pc) o tambien las unidades relativas (em, ex o px)

-Porcentaje: Poner un numero seguido del porcentaje (%), da un valor porciento del tamaño de letra del objeto padre.

-Tamaño relativo: Usa palabras las cuales son relativas al tamaño de letra del objeto padre, ejemplo, larger y smaller.

¿Qué es un XML y para qué sirve?

XML son las iniciales de eXtensible Marup Language.

XML fue diseñado para almacenar y transportar información.

XML fue diseñado para ser ambos leíbles por humanos y por máquinas.

¿Qué características tiene un documento HTML?

Un documento HTML empieza declarando el DOCTYPE: <!DOCTYPE html>

La estructura básica de un document HTML es: <!DOCTYPE html><html><head></head><body></body></html>

HTML permite incrustar CSS y JavaScript dentro del documento HTML. El contenido de estos deberá ser insertado dentro de las etiquetas <style> y <script> respectivamente.

La última versión de HTML es HTML5 y todas las futuras actualizaciones serán en esta misma versión.

HTML permite utilizar diferentes etiquetas dependiendo del contexto como <article>, <aside>, <header>, <footer>, <form>, etc.

¿Que es y para que sirve un archivo XHTML?

  • XHTML (Extensible Hypertext Markup Language) *

Está basado en HTML 4.01, pero sus reglas son mucho más estrictas y versátiles que HTML. XHTML se utiliza para marcar contenido como texto, imágenes y enlaces en forma de hipervínculos para crear una cierta estructura que puede ser mostrada por los navegadores. Los documentos pueden ser estructurados con XHTML para hacerlos legibles para un analizador. El analizador interpreta los elementos de marcado especificados en las definiciones del lenguaje XHTML y reproduce el contenido de estos elementos de una manera específica. Las diferencias importantes entre XHTML y HTML son, por ejemplo, la declaración XML, la definición del tipo de documento, que está en minúsculas, la conformidad exacta con el marco básico y el hecho de que cada elemento abierto debe cerrarse de nuevo. Si una de estas reglas no se adhiere en XHTML, el navegador no puede mostrar el documento, lo que probablemente pueda hacer con HTML.
