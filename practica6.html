Conclusiones

  • Emilio González Rodríguez: En esta práctica utilizamos el elemento canvas para poder dibujar contenido dinámico en la página web. El elemento canvas provee funciones de javascript para dibujar en un lienzo. Es muy interesante ver que los desarrolladores tienen acceso a estas funciones, pues permiten crear aplicaciones dinámicas en lugar de utilizas elementos HTML estaticos.
  • Juan Pablo Amaya García: Se pudo observar por medio de esta practica el elemento Canvas el cual nos permite dibujar por medio de Javascript,cualquier tipo de cosas que se nos pueda ocurrir por medio de sus comandos por ejemplo el Rect el cual dibuja un rectángulo o un cuadrado dependiendo de los datos que le demos e incluso se pueden hacer animaciones
  • Ulises Baltazar Garza Garcia: En esta ocacion se nos pone en practica herramientas de HTML5 que en verciones anteriores no se habían implementado como el podrr colocar videos y musica sin usar otras erramientas externas ademas de hacernos tratar con hipervínculos para hacer que la pagina se conecte con otros sitios aumentando y mejorando nuestra pagina web, ademas nos dieron la libertad para crear la pagina a nuestro plaser dandonos a conoser que las paginas web no tiene un formato predefinido, todo depende la vista que nosotros le demos
  • Jorge Rivera Gomez: En esta practica utilizamos una herramienta mas para el diseño de páginas web CANVAS nos permite aun hacer mas dinámica la página con ayuda de javascrip o algún diseño extra de nuestro gusto y darle una diseño mas particular a nuestro sitio.
